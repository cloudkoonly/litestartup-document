# 價格與方案

專為新創團隊設計的簡單、透明定價。只為您實際使用的部分付費。

## 定價模式

LiteStartup 採用**按郵件計費**模式，而不是按聯絡人計費。這表示您可以匯入無限量聯絡人，只需為您實際寄送的郵件付費。

### 為什麼按郵件計費？

- **省錢**：匯入 10 萬聯絡人，只需為寄送的郵件付費
- **無意外**：基於實際使用量的可預測成本
- **自由擴展**：增加聯絡人名單而不增加成本
- **公平定價**：只為您使用的部分付費

## 計費

### 計費如何運作

1. **月度週期**：計費週期從每月的 1 號到當月最後一天
2. **基本費用**：您的方案月費會在每月 1 號收取
3. **超額費用**：超出方案限額寄送郵件所產生的額外費用
4. **發票**：月底寄送詳細發票
5. **付款**：自動從您的信用卡扣款

### 超額定價

**專業方案**：超過 110,000 封/月後，每 1,000 封郵件 $0.20

**範例：**
- 寄送 120,000 封 = $20 + $2 (10K 超額) = $22
- 寄送 150,000 封 = $20 + $8 (40K 超額) = $28
- 寄送 210,000 封 = $20 + $20 (100K 超額) = $40

### 付款方式

我們接受：
- 信用卡 (Visa, Mastercard, American Express)
- 簽帳卡
- PayPal
- 銀行轉帳（僅限企業方案）

### 發票

- 每月自動產生發票
- 可在您的儀表板中查看
- 寄送到您的帳單信箱
- 可下載為 PDF

### 退款

- 月租訂閱：不提供退款
- 升級：按比例折抵
- 降級：按比例退款或折抵

## 升級與降級

### 升級您的方案

1. 前往 **Settings**（設定） → **Billing**（計費）
2. 點擊 **"Upgrade Plan"**（升級方案）
3. 選擇新方案
4. 檢視定價
5. 點擊 **"Upgrade"**（升級）
6. 變更立即生效

### 降級您的方案

1. 前往 **Settings**（設定） → **Billing**（計費）
2. 點擊 **"Downgrade Plan"**（降級方案）
3. 選擇新方案
4. 確認降級
5. 變更會在計費週期結束時生效

### 按比例計費

當您在月中升級或降級時：
- **升級**：支付剩餘天數的差額
- **降級**：取得剩餘天數的折抵
