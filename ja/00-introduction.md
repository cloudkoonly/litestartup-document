# LiteStartup のご紹介

LiteStartup は、スタートアップ向けに特別に構築されたスマートメールマーケティングプラットフォームです。ビジネスメールサービス（ワークメール）、AI によるコンテンツ生成、そして信頼性の高いメール配信を、手頃な価格で使いやすい 1 つのプラットフォームに統合しました。

## LiteStartup とは？

LiteStartup は、スタートアップを支援する SaaS（Software as a Service）メールプラットフォームです。

- **マーケティングメールの送信**：内蔵の AI コンテンツアシスタントを使用
- **ビジネスメールの管理**：ワークメール機能（Gmail のような）を使用
- **キャンペーンの自動化**：インテリジェントなワークフローを使用
- **購読者リストの構築**：埋め込み可能なサインアップフォームを使用
- **ウェイティングリストの管理**：製品ローンチ前に管理
- **簡単な統合**：REST API を使用して数分で完了

連絡先数ごとに課金される従来のメールマーケティングプラットフォームとは異なり、LiteStartup は送信メール数ごとに課金されます。つまり、無制限の連絡先をインポートでき、実際に送信したメールに対してのみ支払うことになります。

## 主な機能

### Gmail のようなワークメール
ビジネスメールを内蔵した唯一のメールマーケティングプラットフォームです。個別の Google Workspace サブスクリプションは不要です。チームメンバー 1 人あたり年間 72〜144 ドルを節約できます。

### AI コンテンツアシスタント
件名、メール本文、A/B テストのバリエーションを自動的に生成します。作成速度が 10 倍になり、開封率も 10 倍向上します。アイデアから送信までわずか 1 分です。

### 手頃な価格設定
- **無料プラン**：月額 10,000 通
- **Pro プラン**：月額 20 ドルで 110,000 通
- **従量課金**：プラン制限を超えた場合、1,000 通あたり 0.20 ドル
- **連絡先手数料なし**：10 万人の連絡先をインポートしても、支払うのは送信メール分のみ

### 無制限の連絡先
無制限の連絡先をインポートして管理できます。連絡先の保存ではなく、送信したメールに対してのみ支払います。CSV インポート/エクスポートでデータを完全に所有できます。

### エンタープライズインフラストラクチャ
AWS および Google Cloud のエンジニアによって構築されました。99.99% の稼働率、SOC2 および GDPR に準拠しています。スタートアップ価格でエンタープライズレベルの信頼性を提供します。

### スマートオートメーション
- **ウェイティングリスト管理**：メールを収集し、招待コードを送信し、ウェイティングリストを顧客に自動的に変換します
- **購読フォーム**：ダブルオプトイン付きの埋め込み可能なサインアップフォーム、30 秒でどこにでも埋め込めます
- **AI オートメーション**：ユーザー行動によってトリガーされるスマートなワークフロー - ウェルカムシリーズ、再エンゲージメント、ウィンバックキャンペーン

### ベンダーロックインなし
標準的な REST API を使用します。コードを変更することなく、いつでもプロバイダーを切り替えることができます。あらゆるプログラミング言語やフレームワークで動作します。

## 誰が LiteStartup を使うべきですか？

LiteStartup は以下の方に最適です。

- **アーリーステージのスタートアップ**：最初の顧客基盤を構築している企業
- **SaaS 企業**：信頼性の高いトランザクションメールを必要としている企業
- **グロースチーム**：メールマーケティングキャンペーンを実施しているチーム
- **プロダクトチーム**：ウェイティングリストや早期アクセスを管理しているチーム
- **開発者**：複雑さのないシンプルな API 統合を求めている方
- **企業**：ベンダーロックインのない手頃な価格のメールソリューションを探している企業

## LiteStartup の仕組み

### 1. サインアップ
無料アカウントを作成し、月額 10,000 通の無料メールですぐに開始できます。

### 2. API 認証情報の取得
ダッシュボードから数秒で API キーを取得できます。

### 3. 統合
シンプルな REST API を使用して、アプリケーションからメールを送信します。あらゆるプログラミング言語で動作します。

### 4. メールの送信
トランザクションメール、マーケティングキャンペーン、または自動化ワークフローの送信を開始します。

### 5. 追跡と最適化
配信、開封、クリックを監視し、AI の提案を使用してパフォーマンスを向上させます。

| 料金モデル | 送信メールごとの支払い | 連絡先ごとの支払い |
| セットアップ時間 | 5 分 | 30 分以上 |
| API 統合 | ✓ シンプルな REST API | ✓ 複雑な SDK |
| ベンダーロックイン | ✗ なし | ✓ あり |
| 無料枠 | 月額 10,000 通 | 限定的 |
| 稼働率 SLA | 99.99% | 99.9% |

## クイックスタート例

わずか数行のコードで最初のメールを送信できます。

```bash
curl -X POST https://api.litestartup.com/emails/send \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "user@example.com",
    "from": "hello@yourdomain.com",
    "subject": "Welcome to LiteStartup!",
    "html": "<h1>Hello!</h1><p>Welcome to our platform.</p>",
    "text": "Hello! Welcome to our platform."
  }'
```
