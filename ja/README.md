# LiteStartup ドキュメント

LiteStartup プロフェッショナルドキュメント - スタートアップ向けに構築されたスマートメールマーケティングプラットフォーム。

## 概要

LiteStartup は、スタートアップ向けに特別に設計された SaaS メールマーケティングプラットフォームです。ビジネスメール（ワークメール）、AI によるコンテンツ生成、信頼性の高いメール配信を、手頃な価格で使いやすい 1 つのプラットフォームに統合しました。

## 主な機能

✓ **ワークメール** - 組み込みのビジネスメール（Google Workspace は不要）

✓ **AI アシスタント** - 件名とメール本文を生成

✓ **自動化** - ユーザー行動によってトリガーされるスマートなワークフロー

✓ **手頃な価格** - 連絡先数ではなく、送信メール数による支払い

✓ **簡単な統合** - シンプルな REST API、あらゆる言語で動作

✓ **エンタープライズレベルの信頼性** - 99.99% の稼働率、SOC2 および GDPR 準拠
